# Archived Q-Series Scripts

These scripts were used for initial setup and migration.
They are kept here for reference but are no longer needed for daily operations.

- SpyderQ01_Setup.sh - Initial system setup
- SpyderQ02_Dependencies.sh - Dependency installation
- SpyderQ03_InstallGateway.sh - IB Gateway installation
- SpyderQ05_InstallCommands.sh - Command installation
- SpyderQ06_MigrateToQSeries.sh - Migration to Q-series
- SpyderQ35_VerifySystem.sh - System verification

For current functionality, use the consolidated Python modules:
- SpyderQ90_SystemUtilities.py
- SpyderQ91_MonitoringUtilities.py
- SpyderQ92_DiagnosticsUtilities.py
