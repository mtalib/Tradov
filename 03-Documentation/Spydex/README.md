# Spydex Documentation

Spydex is the planned separate application fork of Tradov, intended to run on a different machine with a separate Tradier account after Tradov has cleared the clone-readiness gate.

## Rule of Thumb

Do not start the fork until Tradov has been stable long enough to be a safe baseline. The fork should inherit a known-good release, then be customized for SPX in a separate runtime environment.
