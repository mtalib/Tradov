## APPENDICES

### Appendix A: Technical Glossary

**Sharpe Ratio:** Measure of risk-adjusted return; (return - risk-free rate) / volatility

**Sortino Ratio:** Sharpe ratio using only downside volatility; emphasizes harmful volatility

**Calmar Ratio:** Annualized return divided by maximum drawdown; recovery metric

**Maximum Drawdown:** Largest peak-to-trough decline in portfolio value

**Volatility:** Standard deviation of returns; measure of price fluctuation

**Win Rate:** Percentage of profitable trades

**Profit Factor:** Gross profits divided by gross losses

**Greeks:** Options risk metrics (Delta, Theta, Gamma, Vega)

**Genetic Algorithm:** Machine learning technique using evolutionary principles

**Credit Spread:** Options strategy selling higher premium, buying lower premium

**Fitness Score:** Genetic algorithm performance metric (0-1 scale)
