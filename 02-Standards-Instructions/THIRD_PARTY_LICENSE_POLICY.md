# Third-Party License Policy

Tradov dependency policy:

- Do not add AGPL-licensed libraries.
- Prefer permissive licenses such as MIT, BSD, Apache 2.0, or similarly permissive terms.
- Treat weak copyleft dependencies carefully and review them before adoption.
- If a package license is unclear or cannot be verified, do not add it until reviewed.
- Optional dependencies must follow the same policy, even if they are behind a feature flag or fallback path.

Current pair-trading research stack is intended to remain AGPL-free.
